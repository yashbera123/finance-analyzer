# Finance Analyzer Client

Next.js app router frontend for the finance analyzer.

## Commands

```powershell
npm ci
npm run dev
npm run lint
npm run build
npm run start
```

## Environment

Copy `.env.example` to `.env.local` for local development.

```text
NEXT_PUBLIC_API_URL=http://localhost:8000/api/v1
```

For Vercel, set `NEXT_PUBLIC_API_URL` to the deployed Render API URL including `/api/v1`.
